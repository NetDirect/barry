Device Compatibility List

This list is intended as a rough guide and record of past success when using Barry with various versions of the BlackBerry handheld. It is not guaranteed to be complete, and is based on feedback from users such as yourself.

Please note: A very large majority of BlackBerry devices work with Barry by default, and so a compatibility list like this is somewhat redundant. Please try Barry with your device, and report any issues you may have. Please keep in mind the Known Issues on the main page when testing and reporting.

If you wish to contribute a report, please send it to the mailing list .
